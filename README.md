# Fuck You Pay Me

A sleek web application for tracking work hours, calculating pay, and managing pay periods. Features a dark mode UI, interactive calendar, and dashboard for visualizing earnings.

## Features

* **Dark Mode UI**: A sleek, dark-themed interface for reduced eye strain.
* **Interactive Dashboard**: Displays summaries for the last week, current week, and days until the next payday.
* **Daily Hour Input**: Record hours worked and travel time for each day.
* **Interactive Calendar**: Select dates and visualize days with recorded hours.
* **Pay Period Definition**: Configure pay rate, travel rate, pay period length, and start date.
* **Withholding Configuration**: Define withholding percentages.
* **Automatic Paycheck Calculation**: Real-time calculation of gross pay, withholdings, and net pay.
* **Data Persistence**: All data is saved locally in your browser.

## Getting Started

### Prerequisites

- Node.js (v14.0.0 or higher)
- npm (v6.0.0 or higher)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/EndlessBeard/FuckYouPayMe.git
   cd FuckYouPayMe
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:3000`

### Building for Production

To create a production build:

```bash
npm run build
```

To preview the production build:

```bash
npm run preview
```
